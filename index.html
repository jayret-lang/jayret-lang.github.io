Jayret

Jayret is a hard fork of Pyret with a Java-flavored surface syntax — Java-style braces, types, and declarations, compiling to the same Pyret AST and reusing Pyret's runtime, libraries, and educational ecosystem (the Jayret documentation, DCIC).

  • In-browser playground — write and run Jayret programs without installing anything.
  • VSCode extension — interactive editor for .jrt files.
  • jayret-npm — command-line interface (npm install -g jayret-npm).
  • Source on GitHub — language, npm package, VSCode extension, this site.
